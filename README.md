# 📊 Data Science Salary Analysis

A data analysis project that explores salary trends in the data science industry using Python, data visualization techniques, and Machine Learning. The goal is to understand how factors like experience, job role, location, and company size affect salaries.

---

## 🎯 Project Objective

To analyze global data science salaries and identify key factors that influence compensation, including:
- Experience level  
- Employment type  
- Remote work ratio  
- Company size  
- Geographic location  
- Job title  

---

## 🛠️ Tools & Technologies

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📂 Dataset Overview

The dataset includes:
- Job titles  
- Salary (USD)  
- Experience levels  
- Employment types  
- Remote work ratio  
- Company size  
- Employee residence & company location  

---

## 📊 Key Analyses Performed

- Salary distribution analysis  
- Experience level vs salary comparison  
- Remote work trends  
- Country-wise salary comparison  
- Company size impact on salary  
- Job title salary breakdown  

---

## 🔍 Key Insights

- Senior and executive roles earn the highest salaries  
- Remote work is becoming more common in data science roles  
- Salaries vary significantly across countries  
- Machine Learning Engineers often earn higher averages  
- Larger companies tend to offer more competitive pay
