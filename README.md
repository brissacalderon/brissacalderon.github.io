Data Science Salary Analysis
      This project analyzes data science salaries across different job titles, experience levels, company sizes, and locations using Python and data analysis libraries.
      
Project Overview
      The goal of this project is to explore trends in the data science job market and identify factors that influence salary levels.
  
Key analysis areas include:
    Salary distribution
    Experience level vs salary
    Remote work trends
    Company size impact
    Employee residence analysis
    
Job title comparisons
    Technologies Used
    Python
    Pandas
    NumPy
    Seaborn
    Jupyter Notebook
    Dataset
    
The dataset contains information about:
    Employee salaries
    Job titles
    Experience levels
    Employment types
    Company locations
    Remote ratios    

Features
    Data cleaning and preprocessing
    Exploratory Data Analysis (EDA)
    Data visualization
    Salary trend insights
    Comparative analysis across roles and countries
